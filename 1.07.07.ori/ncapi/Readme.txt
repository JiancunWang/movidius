  
===============================================================================
                       Movidius Neural Compute API
===============================================================================

This directory contains:
   - Redistributable .deb packages for supported target platforms 
   - C and Python3 examples
   - sample networks and sample data
   - tools to download and convert sample trained networks

Please install the Neural Compute Toolkit before using.

Full API documentation is available on http://developer.movidius.com

setup.sh is included that will install deb packages for x86-64 and download
and compile the sample networks needed by the examples.
